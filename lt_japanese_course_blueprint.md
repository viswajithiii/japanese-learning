# Language Transfer Japanese Course Blueprint

> **This document is the master reference for building a DIY Language Transfer-style Japanese course.**
> It serves as a guide for both humans and AI agents working on this project.

## Project Overview

**Goal:** Build a "poor man's" Language Transfer Japanese course continuing from Mihalis's official test tracks 1–20 (available on YouTube). The course follows LT's Thinking Method principles faithfully.

**Status:** Tracks 21–25 are written and playable in the web UI. Tracks 1–20 are Mihalis's official test tracks (not included here, available on the [Language Transfer YouTube channel](https://www.youtube.com/@LanguageTransfer)).

### Project Structure

```
language_transfer/
├── lt_japanese_course_blueprint.md   ← This file. Master plan & methodology reference.
├── tracks_21_25.json                 ← Track data (batch 1). Add more as tracks_26_30.json, etc.
├── build.py                          ← Run after editing any tracks_*.json to regenerate data.js
├── data.js                           ← Auto-generated by build.py. DO NOT edit by hand.
├── index.html                        ← Web UI shell
├── styles.css                        ← UI styling (dark theme, premium design)
└── app.js                            ← UI logic (step-by-step progression, reveals, pronunciation)
```

### Workflow for Adding New Tracks

1. Create a new JSON file (e.g., `tracks_26_30.json`) following the schema in existing files
2. Run `python3 build.py` to regenerate `data.js`
3. Open `index.html` in a browser — no server needed

### Key Decisions Made

- **Format:** Structured JSON with `exposition` and `exercise` step types
- **Language:** All content is in English/romaji. Japanese kana is included in `kana` fields for a "Pronounce" button in the UI (uses Web Speech API)
- **Scope:** Tracks 1–20 are Mihalis's. We author tracks 21–95.
- **Romanization:** Romaji is primary (the learner can't read Japanese). Kana is secondary, for pronunciation reference.
- **No server required:** `build.py` embeds track data directly into `data.js`

## Part 1: The Thinking Method — Detailed Principles

These are the core design rules that govern how every Language Transfer course is built. Any tracks we create must follow them faithfully.

### 1.1 One New Concept Per Exercise ("Isolation of Concepts")

Every sentence the learner is asked to construct introduces **exactly one** new element. Everything else in the sentence uses previously-learned material. This means:

- The learner should be able to *derive* the answer using logic + prior knowledge + one new hint
- If a learner can't produce the sentence, the course is broken — not the learner
- New concepts must "earn their place" — they're only introduced when there's enough prior scaffolding

### 1.2 Structural Panorama First

The course reveals the **broad architecture** of the language before filling in details:

- Core grammatical mechanisms (verb systems, particle logic, word order) are front-loaded
- Traditional textbook ordering (greetings → numbers → colors) is rejected
- The goal: after the course, the learner has a "map" of the language and can continue independently
- Concrete nouns and specialized vocabulary are secondary to structural understanding

### 1.3 Socratic / Inductive Grammar

Grammar is **never** declared as rules to memorize. Instead:

- The teacher poses an English sentence and asks "How would you say...?"
- The learner must *derive* the answer from prior knowledge + one new clue
- Grammatical terminology (subjunctive, conjugation, etc.) is avoided or introduced only *after* the learner has already internalized the concept
- **"Garden Path" technique**: The teacher sometimes deliberately leads toward a predicted common error, then uses the correction as a powerful teaching moment

### 1.4 Masking Repetition

Review is disguised as new learning:

- Previously-learned concepts are embedded into progressively more complex sentences
- The learner practices old material without realizing they're doing "review"
- Each new sentence inherently requires recall of multiple prior concepts
- There are no explicit "review sections" — the architecture itself is the spaced repetition

### 1.5 Tension Contours (Peaks and Valleys)

Cognitive load is deliberately managed:

- **Peaks**: Introduction of challenging new concepts (high cognitive tension)
- **Valleys**: Consolidation periods with simpler applications of new concepts (easing tension)
- This creates a rhythmic learning experience that prevents overwhelm
- Tracks often end on a valley — the learner should feel competent, not confused

### 1.6 Transfer from the Native Language

The method exploits every possible connection between English and the target language:

- **Cognate rules**: Systematic suffix-transformation patterns (e.g., English loanwords in Japanese: doa=door, biru=building)
- **Shared concepts**: Parallel grammatical ideas are highlighted (e.g., "spoken punctuation" for particles)
- **Reframing**: Alien-seeming concepts are explained through familiar English analogies
- For Japanese specifically: English loanwords (katakana vocabulary) are a major resource

### 1.7 Active Recall Through Pausing

- The format is: teacher poses English sentence → learner pauses & constructs → listen to check
- **No note-taking** during lessons (it causes mental "check-out")
- **No memorization** — if you have to memorize, you haven't understood deeply enough
- **Embrace forgetting** — the course will organically revisit; re-deriving is more powerful than never forgetting

### 1.8 Vocabulary Selection

- Structure over vocabulary count — grammatical patterns unlock more communication than word lists
- Vocabulary is introduced *in context*, embedded in sentences that demonstrate structural concepts
- **Cognate-first**: Loanwords and derivable vocabulary before arbitrary new words
- **Functional vocabulary**: New words are introduced because they're needed for a structural teaching point

### 1.9 Error as Data

- Mistakes are not failures — they reveal where the learner's mental model diverges from the language
- Correction is done through re-deriving the logic, not "no, it's X"
- The recorded student makes realistic mistakes, normalizing error for the listener
- Predicted errors are designed into the curriculum as teaching moments

### 1.10 Track Structure

Each track (~5–10 minutes) follows a pattern:

1. Brief bridge from the previous track's concept
2. Introduction of ONE new concept via Socratic questioning
3. Progressive sentence-building incorporating new concept + all prior knowledge
4. Tension easing / consolidation toward the end

## Part 2: Concepts Covered in Test Tracks 1–20

### Tracks 1–8: Foundations

| Concept | Details |
|---|---|
| **Philosophy: Context over translation** | Japanese omits subjects (I, he, she) — inferred from context. Translate *thoughts*, not words |
| **desu (です)** | Universal copula "to be" — same for all subjects (no am/is/are distinction) |
| **Politeness framework** | Balancing politeness with naturalness; cultural sensitivity |
| **Basic adjectives** | ii (good), warui (bad), samui (cold/weather) |
| **Particles as "spoken punctuation"** | ka (か) = spoken question mark; yo (よ) = spoken exclamation mark; to (と) = spoken quotation mark / "with" / "and" |
| **Formal vs. informal speech** | Verbs ending in -masu (polite) vs. plain/dictionary forms |
| **Subject markers wa vs. ga** | wa (は) for topic; ga (が) for emphasis/new information |
| **SOV word order** | Subject-Object-Verb; verb always at end |
| **Word order flexibility** | Order can shift for emphasis as long as verb stays at end |
| **Practical vocabulary** | ramen, nomimono (beverage), tomodachi (friend), food terms |
| **Object marker wo (を)** | Marks the direct object being acted upon |

### Tracks 9–11: Spatial Logic & Possession

| Concept | Details |
|---|---|
| **K/S/D spatial pattern** | Systematic syllable manipulation for question words and spatial indicators |
| **K-words (here/this)** | koko (here), kono (this) |
| **S-words (there/that)** | soko (there), sono (that) |
| **D-words (questions)** | doko (where), dono (which) |
| **Handling objects in sentences** | Sentence construction with object markers |
| **Time expressions** | ashita (tomorrow); time expressions go at beginning of sentence |
| **Contextual omission** | Subjects/objects dropped when already known — Japanese is highly contextual |
| **Particle no (の) for possession** | Functions like 's in English |
| **Complex possessive chains** | Building phrases like "his friend's boyfriend's sake" |

### Tracks 12–15: Verb System & Loanwords

| Concept | Details |
|---|---|
| **no (の) beyond possession** | Connecting concepts: materials (garasu no teeburu = glass table), origins (amerika no hoteru = American hotel) |
| **Verb-final principle** | Japanese word order is flexible *as long as verb stays at end* — helps fight off interruptions |
| **Two verb groups** | "Normal group" (u-ending / godan) and "tabu group" (e-ru or i-ru ending / ichidan) |
| **Deriving masu forms** | How to convert dictionary/plain forms to polite -masu forms for both groups |
| **Adjusting formality** | Shifting register within conversation |
| **Loanwords (gairaigo)** | doa (door), biru (building), terebishon (television) — English words adapted to Japanese phonetics |
| **Cultural loanword split** | Loanwords for Western objects; native Japanese terms for traditional items |
| **Questions via intonation** | Raising intonation as alternative to ka particle |
| **yo (よ) to soften statements** | Using yo to express opinions gently |

### Tracks 16–20: Tense, Suggestions & Demonstratives

| Concept | Details |
|---|---|
| **yo (よ) — emphasis** | "Spoken exclamation mark" (reinforced from earlier) |
| **ne (ね) — shared experience** | "Shared shoes" particle — appeals to mutual understanding |
| **ka (か) vs. ne (ね)** | ka = genuine question; ne = seeking confirmation |
| **Past tense: -mashita** | Formal past tense formation |
| **Negative: -masen** | Formal negative formation |
| **Compound verbs with shimasu** | shimasu (to do) + noun = verb: denwa shimasu (to call), soji shimasu (to clean) |
| **Suggestions: -mashou** | "Let's..." form |
| **Suggestions: -masen ka** | "Why don't we...?" / "Shall we...?" |
| **Full KoSoADo system** | ko- (near speaker), so- (near listener), a- (far from both), do- (question) |
| **Psychological distance** | The KoSoADo pattern extends to psychological/relational distance, not just physical |
| **"Artificial friction"** | Methodology: exercises force active thinking about *why*, not just pattern repetition |
| **"Letting go"** | Over-memorizing causes mistakes; stay relaxed, focus on communicative intent |

## Part 5: Design Notes for Our Course

### What makes Japanese unique for the LT method

> [!IMPORTANT]
> Japanese presents both unique challenges and opportunities compared to the Romance/European languages that LT has completed courses for.

**Challenges:**
- Far fewer cognates than Spanish/French/Italian (but loanwords help!)
- Writing system adds a layer not present in other LT courses
- Politeness levels are structurally embedded, not optional
- Verb groups behave very differently from European conjugation patterns
- Particles have no direct English equivalent

**Opportunities (to exploit via "transfer"):**
- **Loanwords (gairaigo)**: Thousands of English words exist in Japanese — these are our "cognates"
- **Logical regularity**: Japanese conjugation is extremely regular (far fewer irregulars than European languages) — perfect for the "derive, don't memorize" approach
- **Particle system is consistent**: Once you understand particles, they work the same everywhere
- **No grammatical gender, no articles, no plurals**: Major English pain points simply don't exist
- **Context-heavy**: Japanese omits what's obvious — this can be reframed as "Japanese is efficient, not confusing"

### Pacing Guidelines (based on LT completed courses)

- Complete Spanish: ~90 tracks for a much harder grammar system
- Japanese has more regular grammar but more alien structure
- **Target: ~90-95 tracks** seems right
- Phases A-C (verb system + te-form + desire/ability) are the critical core — ~25 tracks
- Phases D-F (conditions + giving + relative clauses) flesh out expressiveness — ~21 tracks  
- Phases G-H (evidentiality + passive/causative) are advanced but essential — ~14 tracks
- Phases I-J (particles + numbers + honorifics) are the "filling in" — ~15 tracks

## Part 6: Track JSON Schema Reference

Each `tracks_*.json` file has this structure:

```json
{
  "courseName": "Language Transfer Japanese",
  "version": "0.1",
  "tracks": [
    {
      "id": 21,
      "title": "Human-readable track title",
      "newConcept": "Short label for the one new concept",
      "conceptNote": "Detailed explanation of the rule for reference",
      "estimatedMinutes": 6,
      "vocabularyIntroduced": [
        { "word": "romaji", "kana": "かな", "meaning": "English meaning", "verbGroup": "ichidan|godan|irregular" }
      ],
      "steps": [
        {
          "type": "exposition",
          "text": "Instructor explanation text"
        },
        {
          "type": "exercise",
          "prompt": "How would you say: ...",
          "hint": "Optional Socratic hint",
          "answer": {
            "romaji": "tabemasen deshita",
            "kana": "たべませんでした",
            "literal": "eat-POLITE-NEG PAST"
          },
          "followUp": "Optional teacher comment after revealing the answer"
        }
      ]
    }
  ]
}
```

**Rules for writing tracks:**

> [!IMPORTANT]
> **EVERY exercise must ask for a FULL SENTENCE, never a bare word or conjugation.**
> Even when the point of the exercise is to practice a new verb form, the prompt must be a complete thought like "How would you say 'Yesterday I didn't drink coffee'?" — never "What's the negative of nomu?" This ensures every exercise simultaneously reinforces prior vocabulary, particles, word order, and tense, achieving "masking repetition" naturally.

- Each track introduces **exactly one** new concept (see Part 1 principles)
- Every exercise answer must be derivable from prior knowledge + the one new concept
- **All exercises are full sentences** — combine the new concept with previously-learned vocabulary, particles, and structures to create natural, complete thoughts
- Exercises should reuse vocabulary and grammar from prior tracks ("masking repetition")
- When introducing a new verb, immediately use it in a sentence with known grammar — don't just ask for the word in isolation
- Aim for 10–14 steps per track (mix of exposition and exercises)
- End each track on a "valley" — consolidation, not confusion
- New vocabulary is introduced via `vocabularyIntroduced` AND organically within exposition steps
